# INVERSE-OF-A-MATRIX
## Aim:
To write a python program to find the inverse of a matrix
## Equipment’s required:

## Algorithm:
### Step1 : First Import Numpy Using import 
### Step 2: Then CO=onvert The List Of Matrix Into Array Using np.array
### Step 3: To find inverse , Use np.linalg.inv()
### Step 4: Then Print The Inverse Matrix

## Program:
```
Program to find the inverse of a matrix.
#Developed by: Surya Prakash S
#RegisterNumber:25014536/212225040443

import numpy as np
A = np.array([[2,1,1],[1,1,1],[1,-1,2]])
b = np.linalg.inv(A)
print(b)
```
